## Как получить API-ключ

1. Перейдите на [exchangerate-api.com](https://app.exchangerate-api.com/sign-up)
2. Зарегистрируйтесь (бесплатный тариф даёт 1500 запросов/месяц)
3. Скопируйте ваш API-ключ и вставьте в переменную `API_KEY` в файле `currency_converter.py`
